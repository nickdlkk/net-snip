# 使用PyWebIO打造的网络剪辑板
数据库使用SQLite

包含密码访问，自动保存，支持MarkDown格式。

## 安装

```bash
pip install -r requirements.txt
```

## 运行

```bash
python main.py
```

## 使用

打开浏览器访问`http://localhost:8080`
